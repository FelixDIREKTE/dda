Könnt ihr beweisen dass die Abstimmungsergebnisse nicht manipuliert sind?

Wir arbeiten daran. Unser Open-Source-Code zeigt, dass die Stimmen korrekt im Server abgegeben und ausgelesen werden. Um ferner zu beweisen dass niemand anders als diese Anwendung Stimmen abgeben kann, planen wir, Stimmen als Blockchain zu speichern. Kurz gesagt bedeutet dass, dass jeder eine verifizierte Kopie anonymisierter Stimmen halten kann und so überprüfen kann dass die Summen mit den Zahlen auf dieser Webseite übereinstimmen.
